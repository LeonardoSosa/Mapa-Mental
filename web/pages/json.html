Características

Los achivos JSON (Javascript Object Notation) tienen un formato que permite el intercambio de información ligero, además están pensados para que un humano pueda leer y escribir la información fácilmente.

A pesar de su nombre, los archivos JSON pueden ser manipulados por la mayoría de lenguajes, teniendo soporte nativo o através de librerías. Algunos ejemplos son : C, C++, C#, Java, JavaScript, Perl, Python, PHP, Go, etc.

Tipos de datos

  • number
  • string
  • boolean
  • object
  • array
  • null

Sintaxis

{ }

para definir objetos

[ ]

para definir arrays

" "

para definir una key (obligatorio)

:

para separar una key de un value

,

para separar datos u objetos

Ejemplos

                    
{
    "name": "eugenia",
    "password": "12345678",
    "adress": 
    {
        "city": "London",
        "street": "Baker Street 123"
    },
    "hobbies": ["run", "code", "eat"],
    "age": 27
}
    

[
    {
        "name": "Juance",
        "id": 123
    },
    {
        "name": "Ernesto",
        "id": 420
    }
]

Observaciones

  • application/json - Mime Type
  • .json - Extensión de archivo
  • Usado en Javascript con AJAX

Útiles

www.jsonlint.com - Comprobar validez archivos JSON

JSON.stringify( ) - Método que convierte un objeto Javascript a string JSON

JSON.parse( ) - Método que convierte un string JSON a objeto Javascript

fetch( ) - Método para hacer resquests de archivos JSON locales o de una api
